**Exercícios de C# - Trilha de Aprendizado .NET**

Este repositório contém os exercícios práticos desenvolvidos durante minha jornada de aprendizado na linguagem C# e na plataforma .NET. 
O objetivo principal é consolidar os fundamentos da programação orientada a objetos e a lógica de desenvolvimento.

- Fundamentos: Variáveis, tipos de dados (int, double, string, bool), entrada e saída de dados no Console.
- Estruturas de Controle: Condicionais estruturadas (if, else if, else) e operadores lógicos.
- Laços de Repetição: Implementação de loops utilizando while e for, incluindo validação de dados com TryParse.
- Lógica de Programação: Resolução de algoritmos aplicados ao cotidiano e cálculos matemáticos.

**Tecnologias Utilizadas**

- Linguagem: C# 12 / .NET 8
- Ambiente de Desenvolvimento: Visual Studio 2022 e Visual Studio Code
- Versionamento: Git e GitHub
